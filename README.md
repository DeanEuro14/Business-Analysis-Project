# Toasters101 Inventory & Sales Tracking System

## Overview
This project focuses on improving inventory and sales tracking processes for Toasters101, a busy coffee shop currently using manual methods to manage stock, sales, and customer tabs.

The proposed solution introduces a simple and affordable digital system using Google Sheets together with the existing Yoco payment setup to improve stock visibility, reduce manual work, and centralise operational tracking.

## Problem
The current process relies heavily on manual tracking using phone notes and physical counting. This creates challenges such as:
- No central source of information
- Frequent stock shortages
- Limited visibility into daily sales
- Customer tabs being tracked informally
- Time spent on manual stock counting

## Objectives
- Improve inventory tracking
- Centralise sales and tab management
- Reduce stock shortages
- Improve operational visibility
- Create a low-cost and easy-to-use solution

## Proposed Solution
A lightweight digital tracking system using:
- Google Sheets
- Existing Yoco payment infrastructure

The system includes:
- Inventory tracking
- Sales recording
- Customer tab management
- Daily reporting
- Low stock alerts
